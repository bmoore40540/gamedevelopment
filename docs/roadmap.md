## Phase 1: Foundation

- [x] Create repository structure
- [x] Add baseline documentation

- [ ] Confirm Android export toolchain on dev machine

## Phase 2: XR Core Loop

- [ ] Create main XR scene

- [ ] Add one grab interactable object
- [ ] Validate app launch on Meta Quest 3S

## Phase 3: Prototype Slice

- [ ] Build a small test room interaction loop

- [ ] Improve scene loading and startup reliability
- [ ] Add basic settings/options menu
- [ ] Profile and optimize draw calls/script hotspots
- [ ] Produce repeatable release APK build process

## Working Definition of Done (Current Milestone)

The current milestone is complete when the app can be exported from Godot,
installed on Meta Quest 3S, and run with stable XR tracking and one working interaction.
